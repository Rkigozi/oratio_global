# Design System

## 1. Purpose

Defines visual consistency and UI behaviour for MVP.

---

## 2. Principles

- minimal
- calm
- readable
- consistent

---

## 3. Colours

- dark background
- soft gradient accents
- white primary text
- muted secondary text

---

## 4. Typography

- clean sans-serif
- clear hierarchy:
  - heading
  - body
  - caption

---

## 5. Components

### Buttons
- rounded
- full width (primary)
- gradient background

### Cards
- rounded
- padded
- readable

### Inputs
- simple
- minimal fields

---

## 6. Interaction

- subtle animations
- clear feedback
- fast response

---

## 7. States

- loading
- empty
- success
- error

---

## 8. Guiding Principle

Design should feel:

calm → clear → effortless
