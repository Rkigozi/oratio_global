# MVP Scope — Oratio

## 1. Purpose

Oratio is a global Christian prayer platform designed to connect people through shared prayer.

---

## 2. MVP Goal

Enable users to:
- explore prayer activity globally
- submit prayer requests
- pray for others
- view personal activity

---

## 3. Core Features

### Map
- global prayer activity view
- aggregated markers
- privacy-safe

---

### Feed
- list of prayer requests
- trending + recent
- category filters

---

### Submit Prayer
- message input (required)
- category (optional)
- location (required)
- anonymous option

---

### Prayer Interaction
- “I Prayed” button
- count increment
- confirmation feedback

---

### Profile
- stats
- activity views:
  - My Prayers
  - Prayed For

---

## 4. Included Enhancements

- translation (on-demand)
- basic moderation safeguards
- safeguarding disclaimer

---

## 5. Out of Scope

- messaging
- notifications
- groups/communities
- monetisation
- advanced moderation

---

## 6. Success Criteria

- users submit prayers
- users interact (“I Prayed”)
- engagement across locations

---

## 7. Guiding Principle

Launch something real, not perfect.
