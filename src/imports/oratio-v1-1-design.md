Design Version 1.1 of a mobile-first UI prototype for an app called Oratio.

Oratio is a global prayer platform where believers can pray for people around the world using an interactive map.

Prayer requests appear as soft glowing lights across a dark world map, representing people asking for prayer.

This version should refine the existing Version 1 design, not redesign the product.

Maintain a calm, minimal, spiritual interface.

Visual Style

The design should feel like entering a quiet digital prayer space.

Visual direction:

dark navy / midnight blue background

minimal interface

soft glowing white lights representing prayer requests

subtle grid or world map outlines

smooth glow effects

The map should resemble a star map of prayer across the world.

Typography

Use:

Headings → Sora

Body text → DM Sans

Typography should feel warm, modern, and easy to read.

Screens to Include
1️⃣ Splash Screen

Display:

ORATIO

A global prayer network

Dark background with subtle glow.

2️⃣ Login / Continue Screen

Options:

Continue with Email
Continue with Google
Continue Anonymously

Minimal and simple.

3️⃣ Global Prayer Map (Home Screen)

The map should take most of the screen.

Top header:

ORATIO
Tap a light to pray for someone

Map elements:

dark world map

glowing prayer lights

lights pulse gently

Lights represent prayer requests.

Map Interaction

Users should be able to:

zoom the map

pan the map

tap lights

Nearby lights should cluster when zoomed out.

Example cluster markers:

72
64
58

Clusters should appear as glowing circles of light.

Map Controls

Include a floating My Location button.

When pressed:

map centers on the user's location
shows nearby prayer requests

Button placement:

bottom-right corner of the map.

4️⃣ Prayer Request Card

When a user taps a prayer light, display a card that slides up from the bottom.

Example:

London, United Kingdom

Pray for my father who is going into surgery tomorrow.

28 people prayed

🙏 I Prayed

Include helper text:

Take a moment to pray for this person.
5️⃣ Prayer Confirmation Screen

After pressing I Prayed, show confirmation.

Example:

🙏 Thank you for praying

Your prayer has been received.

Button:

Return to Map
6️⃣ Submit Prayer Request Screen

Fields:

Prayer message
Name (optional)
Location
Category

Button:

Submit Prayer Request
7️⃣ Prayer Activity Screen

Display global statistics:

Prayers today: 2,842
Countries praying: 22
Total prayers: 12,310

Below show recent prayer requests.

Data Simulation

Include 30–50 sample prayer lights across the map so the interface appears active.

Design Goal

The interface should visually communicate:

“The world slowly lighting up with prayer.”

The experience should feel:

peaceful

reflective

globally connected.
