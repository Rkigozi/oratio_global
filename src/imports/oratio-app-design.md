Design a modern mobile-first app UI for a product called "Oratio".

Oratio is a global prayer platform where believers can pray for people around the world through an interactive map.

Design style:
- calm and spiritual atmosphere
- dark navy / midnight blue background
- soft white glowing lights representing prayer points
- minimal interface
- modern, sleek, peaceful aesthetic
- inspiration from meditation apps and astronomy maps

Primary concept:
A global map where users can see prayer requests and pray for them.

Prayer requests appear as soft glowing dots on the map.

When a user taps a dot, a prayer request card appears.

Screens to design:

1. Home Screen (Global Prayer Map)
- dark interactive world map
- soft white glowing prayer markers
- top navigation with "Oratio"
- bottom action buttons:
  - Pray
  - Submit Prayer Request
  - Prayer Activity

2. Prayer Request Card
- location (city, country)
- prayer request text
- prayer counter
- "I Prayed" button

3. Submit Prayer Request Screen
- text area for prayer request
- optional name field
- location selector
- submit button

4. Prayer Confirmation Screen
- confirmation message: "Thank you for praying"
- updated prayer count
- subtle light animation

5. Global Prayer Activity Screen
- statistics dashboard showing:
  - prayers today
  - countries represented
  - total prayer requests

Design goal:
The interface should feel like a quiet digital prayer space where users feel calm and encouraged to pray for others around the world.
